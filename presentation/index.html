The User Experience

Add a user. Everything else is automatic.

Azure AD user properties drive dynamic security group membership. Set three fields — and the user automatically sees the right hub, the right sites, the right Teams channels.

badge Key Azure AD Properties

These three fields on each user control everything:

apartment
companyName

"Delta Crown Extensions" → joins SG-DCE-AllStaff

work
jobTitle

Contains "Manager/Director/VP" → also joins SG-DCE-Leadership

domain
department

Contains "Marketing" → also joins SG-DCE-Marketing
