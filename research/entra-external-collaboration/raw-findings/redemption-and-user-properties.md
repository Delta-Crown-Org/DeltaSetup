# Raw findings: invitation redemption and user properties

Key excerpts captured from Microsoft Learn:
- Invited B2B users start with consent status / invitation state of PendingAcceptance / Pending acceptance.
- After accepting, the invitation becomes accepted and the identity provider details are updated.
- Automatic redemption reduces first-run consent prompts, but only when both partner tenants enable it.
- Direct links still require tenant context and can still be affected by app/policy configuration.
