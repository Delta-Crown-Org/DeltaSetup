# Raw findings: cross-tenant synchronization

Key excerpts captured from Microsoft Learn:
- Cross-tenant sync automates creating, updating, and deleting Entra B2B collaboration users and groups across tenants.
- Users created by cross-tenant sync have the same Teams and Microsoft 365 experience as manually invited B2B collaboration users today.
- Users can be synchronized to target tenants as external members (default for some scenarios) or external guests.
- Cross-tenant sync does not migrate SharePoint/OneDrive data and is not a tenant migration tool.
