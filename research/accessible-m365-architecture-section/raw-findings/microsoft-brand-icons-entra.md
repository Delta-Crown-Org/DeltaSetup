# Raw findings: Microsoft product terminology, icons, and trademarks

## Microsoft Entra overview

Source: https://learn.microsoft.com/en-us/entra/fundamentals/what-is-entra

Extracted findings:

- Microsoft Entra is a family of identity and network access products supporting Zero Trust.
- Microsoft Entra ID is the foundational product; a cloud identity and access management service for authentication, policy enforcement, and protection for users, devices, apps, and resources.
- Microsoft 365, Azure, and Dynamics CRM Online subscribers are already using Microsoft Entra ID; every tenant is automatically a Microsoft Entra tenant.
- Last updated: 2026-04-09.

## Microsoft Entra architecture icons

Source: https://learn.microsoft.com/en-us/entra/architecture/architecture-icons

Extracted findings:

- Architecture diagrams can communicate design decisions and relationships between workload components.
- Microsoft provides official Microsoft Entra ID icons for architectural diagrams, training materials, or documentation.
- Do: use icons to illustrate how products can work together; include product name near the icon.
- Don’t: crop, flip, rotate, distort, change icon shape, use Microsoft product icons to represent your product/service, or use icons in marketing communications.
- Terms: Microsoft permits copying, distributing, and displaying icons only for the permitted use unless explicit permission is granted; Microsoft reserves all other rights.
- Last updated: 2023-10-23.

## Microsoft trademark and brand guidelines

Source: https://www.microsoft.com/en-us/legal/intellectualproperty/trademarks

Extracted findings:

- Microsoft brand assets include trademarks, logos, icons, designs, trade dress, fonts, product/service names, sounds, emojis, and other brand features.
- Many uses, including logos, app/product icons, and designs, require a license first.
- Wordmarks and product names may generally be used truthfully to describe compatibility or integration if not misleading and not more prominent than the user’s own brand.
- Do not imply affiliation, endorsement, sponsorship, or approval by Microsoft.
- Do not use Microsoft logos, icons, or designs without permission.
- Include a trademark footnote when using Microsoft trademarks.
